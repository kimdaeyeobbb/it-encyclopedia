<hr>

#### 대엽

- 답변

#### 민지

- 답변

#### 다은

- 답변

#### 선아

- 부수 효과가 있는 연산자에는 할당 연산자 = 과 증가 연산자 ++, 감소 연산자 --, 그리고 delete 연산자가 있습니다. 할당 연산자는 변수 값이 변하는 부수 효과가 있고, 증가/감소 연산자는 피연산자의 값을 변경하는 부수 효과가 있고, delete 연산자는 객체의 프로퍼티를 삭제하는 부수 효과가 있습니다.

<hr>

#### 민지
